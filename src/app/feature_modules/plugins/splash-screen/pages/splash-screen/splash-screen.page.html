Splash Screen

Splash Screen works at the beginning of the app. You can set extended splash screen if the app takes a long time to load. You'll see similar animation at the beginning of this Full App.

 Set a Splash Screen ON
SplashScreen.show()
Set a Splash Screen ON with Timer 
SplashScreen.show({{'{'}} 
  showDuration: 2000 
{{'}'}})

You can show such splash in middle of the app to hide any processing etc. as well

 Show Splash Screen for 2s Set a Splash Screen OFF
SplashScreen.hide()

You can find remaining Splash Screen options in documentation, which are also added in capacitor.config.ts in root folder
