social share

Capacitor Social Share API provides methods for sharing content in any sharing-enabled apps the user may have installed.

You can share three types of data - Title, text message and URL. The URL can be a file URL as well.

Different apps can behave different in terms of the data they allow to be pre-filled

For extended functionality, check the Cordova Social Sharing implementation
